# BatchCard Component

A fully composable, accessible batch card component following modern React patterns.

## Features

✅ **Composable Architecture** - Break down into smaller, reusable parts  
✅ **TypeScript First** - Full type safety with proper prop types  
✅ **Variant System** - CVA-powered variants for different sizes  
✅ **Polymorphic** - Support for `asChild` prop using Radix Slot  
✅ **Accessible** - ARIA attributes, keyboard navigation, focus states  
✅ **Flexible** - Works as link, button, or static card  
✅ **ClassName Extensible** - Override styles with `cn()` utility  

## Installation

The component uses these dependencies:
```bash
npm install class-variance-authority clsx tailwind-merge @radix-ui/react-slot
```

## Basic Usage

```tsx
import {
  BatchCard,
  BatchCardWrapper,
  BatchCardImage,
  BatchCardContent,
  BatchCardHeader,
  BatchCardTitle,
  BatchCardMeta,
  BatchCardMetaItem,
} from '@/components/BatchCard';

// As a link
<BatchCardWrapper href="/batch/123" aria-label="View NEET batch">
  <BatchCard>
    <BatchCardImage alt="Educators" />
    <BatchCardContent>
      <BatchCardHeader language="En" topic="FULL SYLLABUS" />
      <BatchCardMeta>
        <BatchCardTitle>NEET 2022 Batch</BatchCardTitle>
        <BatchCardMetaItem>Starts in 3 days</BatchCardMetaItem>
        <BatchCardMetaItem>Morning classes</BatchCardMetaItem>
        <BatchCardMetaItem>Expert faculty</BatchCardMetaItem>
      </BatchCardMeta>
    </BatchCardContent>
  </BatchCard>
</BatchCardWrapper>

// As a button
<BatchCardWrapper 
  onClick={() => console.log('Clicked')}
  aria-label="View batch details"
>
  <BatchCard size="compact">
    {/* ... */}
  </BatchCard>
</BatchCardWrapper>

// Static card with custom styling
<BatchCard size="wide" className="shadow-lg hover:shadow-xl">
  {/* ... */}
</BatchCard>
```

## Component API

### BatchCard

Main container for the card.

**Props:**
- `size?: "default" | "compact" | "wide"` - Card size variant
- `className?: string` - Additional classes
- `asChild?: boolean` - Use Slot for composition
- Extends `React.ComponentProps<"article">`

### BatchCardWrapper

Interactive wrapper for navigation or actions.

**Props:**
- `href?: string` - Renders as link
- `onClick?: () => void` - Renders as button
- `interactive?: boolean` - Enable hover/focus states
- `className?: string` - Additional classes
- `asChild?: boolean` - Use Slot for composition
- Extends `React.ComponentProps<"div">`

### BatchCardImage

Displays educator thumbnails.

**Props:**
- `images?: string[]` - Array of image URLs (max 2 shown)
- `alt?: string` - Alt text for images
- `className?: string` - Additional classes
- Extends `React.ComponentProps<"div">`

### BatchCardContent

Container for card text content.

**Props:**
- `className?: string` - Additional classes
- Extends `React.ComponentProps<"div">`

### BatchCardHeader

Language badge and topic tag.

**Props:**
- `language?: string` - Language code (e.g., "En", "Hi")
- `topic?: string` - Topic/category text
- `className?: string` - Additional classes
- Extends `React.ComponentProps<"div">`

### BatchCardTitle

Main heading for the batch.

**Props:**
- `className?: string` - Additional classes
- `asChild?: boolean` - Use Slot for composition
- Extends `React.ComponentProps<"h3">`

### BatchCardMeta

Container for metadata items.

**Props:**
- `className?: string` - Additional classes
- Extends `React.ComponentProps<"div">`

### BatchCardMetaItem

Individual metadata row with icon.

**Props:**
- `icon?: string` - Icon image URL
- `className?: string` - Additional classes
- Extends `React.ComponentProps<"div">`

## Accessibility

- ✅ Keyboard navigation (Tab, Enter, Space)
- ✅ ARIA labels for screen readers
- ✅ Focus visible states (`focus-visible:ring`)
- ✅ Semantic HTML elements
- ✅ Role attributes for interactive elements

## Customization

```tsx
// Override styles with className
<BatchCard className="border-2 border-primary">
  {/* ... */}
</BatchCard>

// Use asChild for composition
<BatchCardWrapper asChild>
  <Link to="/batch/123">
    <BatchCard>{/* ... */}</BatchCard>
  </Link>
</BatchCardWrapper>

// Custom images
<BatchCardImage 
  images={[img1, img2, img3]} 
  alt="Top educators"
  className="h-[200px]"
/>
```

## Data Attributes

Each component has a `data-slot` attribute for CSS targeting:

```css
[data-slot="batch-card"] { /* ... */ }
[data-slot="batch-card-wrapper"] { /* ... */ }
[data-slot="batch-card-image"] { /* ... */ }
```

## Notes

- Components follow the composition pattern from shadcn/ui
- All components use `React.forwardRef` for ref forwarding
- Variants are powered by CVA (Class Variance Authority)
- ClassName merging uses `cn()` utility (clsx + tailwind-merge)
